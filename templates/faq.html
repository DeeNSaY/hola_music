🎵 О сервисе Hola Music

Что такое Hola Music и для кого он? ▼

Hola Music — это инновационный веб-сервис для анализа музыкальных чартов с использованием искусственного интеллекта. Сервис создан для:

  • 🎵 Музыкальных энтузиастов, которые хотят глубже понимать музыку
  • 📊 Исследователей музыкальных трендов
  • 🎓 Студентов, изучающих музыкальную индустрию
  • 💡 Просто любознательных людей, интересующихся музыкой

Наша миссия — сделать анализ музыки доступным и понятным для каждого.

Откуда берутся данные о треках? ▼

Данные поступают из официального API Яндекс Музыки. Мы получаем:

  • 📋 Актуальный чарт (топ треков)
  • 🎤 Названия треков и исполнителей
  • ⏱️ Длительность
  • 🖼️ Обложки альбомов
  • 📝 Тексты песен (когда доступны)

Все данные кешируются в нашей базе данных для быстрой загрузки.

Как часто обновляется чарт? ▼

Чарт обновляется автоматически каждый раз при загрузке страницы, но используется кеширование для ускорения работы. Вы также можете принудительно обновить данные через API (только для авторизованных пользователей). Обычно чарт Яндекс Музыки обновляется ежедневно, поэтому вы всегда видите актуальную информацию.

🤖 Hola AI — ваш музыкальный эксперт

Что умеет Hola AI? ▼

Hola AI на базе нейросети DeepSeek умеет:

  • 📊 Анализировать популярность треков — объясняет, почему песня стала хитом на основе BPM, тональности, структуры
  • 🎵 Объяснять музыкальные термины — простым языком рассказывает о BPM, тональности, энергичности, танцевальности
  • 🌍 Рассказывать о трендах — анализирует текущие музыкальные тенденции
  • 💡 Давать рекомендации — советует похожих исполнителей и треки
  • 📝 Анализировать тексты — помогает понять смысл и структуру песен
  • 🎯 Сравнивать треки — находит сходства и различия между композициями
Какая нейросеть используется? ▼

Мы используем DeepSeek API — мощную нейросеть, специализирующуюся на анализе данных и генерации текста. DeepSeek обучена на огромном массиве информации, включая музыкальную аналитику, и способна давать экспертные, глубокие и при этом понятные ответы. Нейросеть работает в режиме реального времени и постоянно совершенствуется.

Как правильно задавать вопросы, чтобы получить лучший ответ? ▼

Вот несколько советов для эффективного общения с Hola AI:

  • ✅ Будьте конкретны — вместо "расскажи о музыке" спросите "почему трек X популярен?"
  • ✅ Указывайте трек или исполнителя — AI знает текущий чарт, упоминайте названия
  • ✅ Задавайте уточняющие вопросы — если ответ неполный, спросите подробнее
  • ✅ Спрашивайте о сравнениях — "сравни трек A и трек B по энергичности"
  • ✅ Интересуйтесь музыкальной теорией — "как BPM влияет на восприятие песни?"

Примеры хороших вопросов: "Почему песня Espresso стала вирусной в TikTok?", "Какой BPM у трека Lose Control и почему это важно?", "Сравни тональность треков в топ-5 чарта".

🔐 Аккаунт и безопасность

Как зарегистрироваться? ▼

Регистрация занимает меньше минуты:

  1. Нажмите кнопку "Войти" в правом верхнем углу
  2. Выберите "Зарегистрироваться"
  3. Укажите имя пользователя (будет отображаться в профиле)
  4. Введите email (используется для входа)
  5. Придумайте надёжный пароль (минимум 6 символов)
  6. Подтвердите пароль и нажмите "Зарегистрироваться"

После регистрации вы автоматически войдёте в аккаунт и сможете пользоваться всеми функциями, включая чат с AI и сохранение истории диалогов.

Как происходит шифрование паролей? ▼

Мы используем современные стандарты безопасности:

  • 🔒 Алгоритм хеширования: pbkdf2:sha256 (рекомендованный Flask стандарт)
  • 🧂 Соль (salt): генерируется автоматически для каждого пароля
  • 🔄 Итерации: множественное хеширование для усиления защиты
  • 🚫 Обратное преобразование невозможно: пароль нельзя восстановить из хеша

Даже при утечке базы данных злоумышленники не смогут получить ваши пароли. Мы никогда не храним пароли в открытом виде.

Что хранится в моём профиле? ▼

В вашем профиле хранится:

  • 👤 Публичная информация: имя пользователя, email, дата регистрации
  • 💬 История чатов с Hola AI: все ваши диалоги (доступны только вам)
  • 🎵 Просмотренные треки: для персонализации рекомендаций (в разработке)

Вы можете просмотреть всю историю в разделе "Профиль". Данные не передаются третьим лицам и используются только для работы сервиса.

Как удалить свой аккаунт? ▼

Для удаления аккаунта свяжитесь с нами через GitHub или email, указанный в подвале сайта. Мы удалим все ваши данные в течение 48 часов. Обратите внимание, что это действие необратимо — история чатов и сохранённые данные будут потеряны навсегда.

⚙️ Технические вопросы

На каких устройствах работает сайт? ▼

Сайт полностью адаптивен и отлично работает на:

  • 💻 Компьютерах — Windows, macOS, Linux (любые браузеры)
  • 📱 Смартфонах — iOS, Android (Chrome, Safari, Firefox)
  • 📟 Планшетах — iPad, Android-планшеты

Рекомендуем использовать последние версии браузеров для лучшей производительности.

Какие технологии используются на сайте? ▼

Стек технологий Hola Music:

  • Backend: Python 3.11, Flask, SQLAlchemy
  • База данных: PostgreSQL (на Render) / SQLite (локально)
  • Frontend: HTML5, CSS3 (Spotify-стиль), JavaScript (vanilla)
  • API: Yandex Music API, DeepSeek API
  • Деплой: Render.com, Gunicorn
  • Аутентификация: Flask-Login, Werkzeug Security

Весь код открыт и доступен на GitHub.

Почему некоторые тексты песен недоступны? ▼

Тексты песен предоставляются через API Яндекс Музыки. Если текст недоступен, это может быть связано с:

  • 📜 Отсутствием лицензированных текстов у правообладателя
  • 🔒 Ограничениями API Яндекс Музыки
  • 🆕 Новизной трека (текст может появиться позже)

В таких случаях Hola AI всё равно может ответить на вопросы о треке, используя другие метаданные (BPM, тональность, популярность).

🛠️ Для разработчиков

Есть ли открытое API? ▼

Да, наш API доступен для разработчиков. Основные эндпоинты:

  • POST /api/chat — общение с Hola AI (требуется авторизация)
  • POST /api/refresh-chart — принудительное обновление чарта
  • GET /api/chart — получение текущего чарта (в планах)

Для доступа к API необходимо зарегистрироваться и использовать токен авторизации. Подробная документация появится позже.
